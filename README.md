# A Simple Bank System
#### 主要有如下功能：
- [创建一个新账户](#user-content-创建一个新账户)
- [查询账户信息](#user-content-查询账户信息)
- [修改账户信息](#user-content-修改账户信息)
- [存款](#user-content-存款)
- [取款](#user-content-取款)
- [销毁账户](#user-content-销毁账户)
- 把账户信息储存在txt文档中

#### 创建一个新账户
- 输入姓名、身份证号、密码（再次输入确认密码）、预存金额（不少于100元）
- 信息存入字典
- 输出随机生成的6位卡号

#### 查询账户信息
- 登录
- 输出姓名、卡号、身份证号、余额

#### 修改账户信息
- 修改账户名称
- 修改身份证号
- 修改密码（重复输入确认密码）

#### 存款
- 输入金额
- 打印余额

#### 取款
- 输入金额（不多于余额）
- 打印余额

#### 销毁账户
- 确认卡号
- 确认密码
- 从字典中删除账户
